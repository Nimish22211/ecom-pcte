# public/

Static files served as-is by Vite. Put your favicon, robots.txt, or any file that needs a fixed public URL here.

Files here are accessible at the root URL — e.g. `public/favicon.ico` is served at `http://localhost:5173/favicon.ico`.

Do NOT import files from here in your JS/JSX. Use `src/assets/` for images you import in components.
